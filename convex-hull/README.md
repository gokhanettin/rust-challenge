# CONVEX HULL

This project implements popular [convex hull algorithms][1].

- [x] Gift Wrapping
- [ ] Graham scan 
- [ ] Quickhull
- [ ] Divide and conquer
- [ ] Monotone chain 
- [ ] Incremental convex hull algorithm
- [ ] The ultimate planar convex hull algorithm 
- [ ] Chan's algorithm
